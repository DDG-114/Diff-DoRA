# Electric Load / Charging / Renewable Transfer Project

This workspace archives the earlier EV charging demand, Wotai energy-system, and renewable-to-Wotai transfer experiments.

## Contents

- `docs/`: experiment notes, preprocessing docs, ablation plans, transfer workflow notes, and audit reports.
- `scripts/`: legacy helper scripts and run scripts for ST-EVCDP, UrbanEV, renewable generation, Wotai, ablations, and visualization.
- `outputs/`: saved adapters, evaluation JSON files, and visualization artifacts from the earlier load/charging/PV experiments.
- `references/`: papers, extracted figures, and other reference files used by the earlier project.

## Notes

The shared source code remains in the repository-level `src/` package. Top-level `outputs/...` entries are kept as symlinks for compatibility with older commands and notebooks.
